Full-stack mini blog project with an Express.js REST API (backend) and a simple frontend server. Supports CRUD operations on blog posts with an in-memory data store.

first install node.js
Install dependencies: npm install
Running the Project:
Backend (API):node index.js
Frontend:node frontendserver.js

Endpoints

GET /posts → Get all posts

GET /posts/:id → Get a specific post (when update)

POST /posts → Create a new post

PATCH /posts/:id → Update a post

DELETE /posts/:id → Delete a post
